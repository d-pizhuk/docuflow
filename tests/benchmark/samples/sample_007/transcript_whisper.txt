In this tutorial I'm going to show you how to write a simple Python script using a text editor and then run it from the terminal. This is a fundamental skill and we'll go through it step by step so everything is clear. First, open your terminal. Let's create a dedicated folder for our script. Type mkdir python demo and press enter. Then navigate into it with cd python demo and press enter. Now let's create a python file. On Windows type notepad hello.py and press Enter. Notepad will ask if you want to create a new file. Click Yes. On Mac you can type nano hello.py or open it in any text editor you prefer. Now in the text editor let's write our script. On the first line type print followed by parenthesis and inside the parenthesis type the string hello world in quotes. So it should look like print, open parenthesis, quote, hello world, quote, close parenthesis. Let's add a few more lines to make it more interesting. On the next line, let's declare a variable. Type name equals and then your name in quotes. For example, name equals Alice. On the line, I'll... that type print open parenthesis f dash string with f before the quotes then hello comma space and then curly braces with name inside then exclamation mark close quote so that the line reads printf! Now let's add a simple loop Type 4, I, in range, open parenthesis, 5, close parenthesis, column and press ENTER. on the next intended line type print open parenthesis f-string f iteration number curly brace i plus one close curly brace close quote close parenthesis. Make sure that line is intended with four spaces because Python uses indentation to define code blocks. Now save the file. In Notepad press Ctrl S. In Nano press Ctrl O to save and Ctrl X to exit. Now switch back to your terminal. Make sure you are in the Python demo folder. Type python hello.py and press enter. If you are on a system where Python refers to Python 2, try Python 3. You should see the output of your script in the terminal. The first line should say hello world, the second line should say hello Alice with whatever name you used. Then you should see iteration number 1 through iteration number 5 from the loop. If you see a syntax error or indentation error, check your script for any typos or missing indentation. Congratulations, you have just written and executed your first Python script from the command line.
