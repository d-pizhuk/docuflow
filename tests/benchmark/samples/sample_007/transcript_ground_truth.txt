In this tutorial I'm going to show you how to write a simple Python script using a text editor and then run it from the terminal. This is a fundamental skill and we'll go through it step by step so everything is clear. First open your terminal. Let's create a dedicated folder for our script. Type mkdir python-demo and press Enter. Then navigate into it with cd python-demo and press Enter. Now let's create the Python file. On Windows type notepad hello.py and press Enter. Notepad will ask if you want to create a new file, click Yes. On Mac you can type nano hello.py or open it in any text editor you prefer. Now in the text editor let's write our script. On the first line type print followed by parentheses and inside the parentheses type the string Hello, World! in quotes. So it should look like print open parenthesis quote Hello, World! quote close parenthesis. Let's add a few more lines to make it more interesting. On the next line let's declare a variable. Type name equals and then your name in quotes, for example name equals Alice. On the line after that type print open parenthesis f-string with f before the quotes, then Hello comma space and then curly braces with name inside, then exclamation mark, close quote, close parenthesis. So that line reads print f Hello name exclamation mark. Now let's add a simple loop. Type for i in range open parenthesis 5 close parenthesis colon and press Enter. On the next indented line type print open parenthesis f-string f Iteration number curly brace i plus 1 close curly brace close quote close parenthesis. Make sure that line is indented with four spaces because Python uses indentation to define code blocks. Now save the file. In Notepad press Control S. In nano press Control O to save and Control X to exit. Now switch back to your terminal. Make sure you are in the python-demo folder. Type python hello.py and press Enter. If you're on a system where python refers to Python 2, try python3 hello.py instead. You should see the output of your script in the terminal. The first line should say Hello, World! The second line should say Hello Alice with whatever name you used. Then you should see Iteration number 1 through Iteration number 5 from the loop. If you see a SyntaxError or IndentationError check your script for any typos or missing indentation. Congratulations you have just written and executed your first Python script from the command line.
