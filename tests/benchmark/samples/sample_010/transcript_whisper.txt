In this tutorial I'm going to show you how to write a docker file. Build a docker image from it and then run your application as a docker container. Docker is a tool that packages your application together with all its dependencies into a portable container that runs the same way on any machine. So let's assume we have a simple Python Flask application. The project folder contains a file called app.py. which is our application and a requirements.txt with the dependencies. The app.pipe just starts a flask server that returns hello from docker on the home root. The first thing we need to do is create a docker file in the project route. Open your text editor and create a new file called docker file. with a capital D and no file extension the first instruction in a docker file is always FROM which specifies the base image. on the first line. This tells Docker to use the official Python 3.11 slim image as the starting point. The slim variant is smaller and faster to download than the full image. On the next line type workdir This sets the working directory inside the container to slash app. All subsequent commands will run from this directory. Now type copy requirements.txt with a dot at the end. The dot means copy into the current working directory, which is slash app. on the next line run pip install no cache dir requirements txt This installs the Python dependencies inside the container. We do this before copying the application code because Docker caches each layer and if dependencies don't change, this step won't rerun on future bits. Now type copy dot dot to copy the rest of the application code into the container. Then type expose5000 to tell Docker that the container listens on port 5000. Finally type cmd with square brackets, then the command to start the application. So cmd open square bracket quote python quote comma quote app dot pi quote close square bracket Save the Docker file. Now open your terminal and navigate to the project folder. Type docker build my flask app dot and press enter The T flag gives the image a name, in this case my flask app, and the dot tells Docker to look for the Docker file in the current directory. Docker will now execute each instruction in the Docker file and you will see output for each step. This might take a minute the first time because it needs to download the base image. When the build finishes you will see a message saying successfully build followed by an image ID and then successfully tagged my flask app latest Now let's run the container. Type docker run dash p 5000 5000 my flask app and press enter. the "-P50005000 flag maps port 5000 on your host machine to port 5000 inside the container You should see the Flask development server starting up. Open your browser and go to HTTP localhost 5000 You should see the message Hello from Docker in your browser. Your Flask application is now running inside a Docker container. To stop the container press Ctrl-C in the terminal.
